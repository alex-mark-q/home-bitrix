# JS Extension: Custom Leads Kanban

### 1. Описание
Расширение для модификации отображения Канбана в CRM. Подсвечивает колонку «Конвертировано» (CONVERTED) для визуального контроля завершенных лидов.

### 2. Функционал
* Отслеживает событие отрисовки Канбана `Kanban.Grid:onRender`.
* Находит колонку с идентификатором `CONVERTED`.
* Добавляет красную рамку (`2px solid red`) к контейнеру элементов колонки.

### 3. Структура
* `config.php` — регистрация расширения и подключение JS-файла.
* `src/main.js` — логика манипуляции DOM-деревом через `BX.addCustomEvent`.

### 4. Подключение
Для активации расширения на странице используйте:
```php
\Bitrix\Main\UI\Extension::load('work.custom_leads');
